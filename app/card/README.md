# عبايات — بطاقات المنتجات

مشروع Next.js + Tailwind CSS لعرض بطاقات المنتجات بتصميم عربي RTL.

## 🚀 تشغيل المشروع

```bash
# تثبيت الحزم
npm install

# تشغيل بيئة التطوير
npm run dev
```

ثم افتح [http://localhost:3000](http://localhost:3000) في المتصفح.

## 📁 هيكل الملفات

```
├── app/
│   ├── globals.css          # CSS عام + Tailwind
│   ├── layout.tsx           # Layout مع خط عربي IBM Plex Sans Arabic
│   └── page.tsx             # صفحة الشبكة مع فلتر + ترتيب
│
├── components/
│   └── ProductCard.tsx      # بطاقة المنتج (الكومبوننت الرئيسي)
│
└── data/
    └── products.ts          # بيانات المنتجات الستة
```

## ✨ المميزات

- **RTL كامل** — دعم اللغة العربية بالكامل
- **Responsive** — 2 أعمدة على الموبايل، 3 على التابلت، 4 على الديسكتوب
- **فلترة وترتيب** — فلتر حسب النوع وترتيب حسب السعر والتقييم
- **Wishlist** — زر القلب مع تأثير بصري
- **اختيار الألوان** — نقاط الألوان على الصورة
- **اختيار المقاسات** — مع إظهار النفاد
- **شريط المخزون** — لإظهار الكمية المتبقية
- **إضافة سريعة** — Overlay عند hover على الصورة
- **تأكيد الإضافة** — تغيير لون الزر عند الضغط
- **خط IBM Plex Sans Arabic** — لقراءة ممتازة بالعربية

## 🎨 التخصيص

لتغيير المنتجات، عدّل ملف `data/products.ts`.

## 🛠 التقنيات

- **Next.js 14** (App Router)
- **Tailwind CSS 3**
- **TypeScript**
- **IBM Plex Sans Arabic** (Google Fonts)
